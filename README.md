# FakeSeg : Prior-Aware Vision-Language Prompting for Image Forgery Localization

## 📌 Introduction
This repository provides a preliminary implementation of **FakeSeg**, a framework for image forgery localization.

⚠️ **Note:**  
The current version of the code is **for testing purposes only** and does not include the full implementation described in the paper.

The complete codebase, including training and evaluation pipelines, will be **fully released upon paper acceptance**.
